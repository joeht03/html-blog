Week 1: Initial Setup and Literature Review

On my first day, I attended the department meeting where I met the current PhD students I'd be working alongside. They each explained what they were currently working on and their plans for the week/going forward. After this meeting, I was kindly shown around the lab setup, had relevant content explained to me and was sent reading material to review. At this moment, my project goal is not set in stone, but I think I will work on data analysis and image formulation from scan data.

I started working on some python code to visualise some data collected. Using the process explained to me by PhD student Ben, I put together some code for taking the detected THz pulse data and isolating the relevant returned information, then analysing that through use of Fourier Transforms.
